# Java ATM System (JDBC)

## Overview
This project is a simple ATM system implemented in Java. It demonstrates basic banking operations such as account creation, balance inquiry, deposit, withdrawal, and transaction history. The system uses JDBC for database connectivity.

## Features
- User authentication
- Account creation
- Balance inquiry
- Deposit and withdrawal
- Transaction history
- Secure database operations via JDBC

## Requirements
- Java (JDK 8 or higher)
- JDBC-compatible database (e.g., MySQL, PostgreSQL)
- Database driver (e.g., MySQL Connector/J)

## Setup Instructions
1. Clone or download this repository.
2. Set up your database and create the required tables.
3. Update the database connection details in `ATM.java`.
4. Compile the project:
	```
	javac ATM.java
	```
5. Run the application:
	```
	java ATM
	```

## Usage
- Follow the on-screen prompts to log in or create an account.
- Perform banking operations as needed.

## Customization
- Modify `ATM.java` to add new features or change database logic.
- Update SQL queries as per your database schema.
